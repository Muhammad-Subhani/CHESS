# ♟️ CHESS

A classic, interactive Chess game built to run directly in the browser. 

---

## 🚀 Features

* **Full Chess Logic:** Supports standard chess moves, turn-taking, and piece interactions.
* **Move Validation:** Highlights valid paths for selected pieces and prevents illegal moves.
* **Game State Detection:** Automatically tracks and displays game status (Check, Checkmate).
* **Responsive UI:** A clean board designed entirely with HTML and CSS.

---

## 🛠️ Tech Stack

* **HTML5:** Structure of the chess board and UI elements.
* **CSS3:** Styling, board layout, and smooth visual animations.
* **JavaScript (Vanilla):** Game logic, move validation, and piece movement handlers.
